# Animotion CSS

Generador visual de animaciones CSS para crear transiciones suaves, microinteracciones y efectos complejos sin pelearte con keyframes a mano.

- Features
  - Editor visual de keyframes con previsualización en vivo
  - Presets de animaciones comunes (fade, slide, bounce, scale, etc.)
  - Export de código CSS listo para pegar en tu proyecto

- Stack
  - React
  - TypeScript
  - Tailwind CSS
